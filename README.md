# 📈 基于 XGBoost 的多因子选股策略（沪深300）

<p align="center">
  <b>量化投资 | 多因子模型 | 机器学习 | 策略回测</b><br>

</p>

---

## 📌 一、项目简介

本项目基于沪深300成分股（HS300），构建了一套**多因子选股策略系统**，并引入机器学习模型（XGBoost）预测股票未来一个月收益率，从而实现超额收益（Alpha）挖掘。

项目完整实现了量化研究的核心流程：

> 因子构建 → 数据预处理 → 特征工程 → 模型训练 → 收益预测 → 策略构建 → 回测评估
> 
---

## 🧠 二、核心研究思路

整体逻辑如下：

1. 构建多因子体系（技术 + 基本面 + 行为）
2. 对因子进行去极值、标准化和中性化处理
3. 使用 XGBoost 回归模型预测未来收益率（fwd_return）
4. 每月选取预测收益最高的股票构建组合
5. 进行历史回测并评估策略表现

## 📊 四、因子体系（15个核心因子）

本项目采用多维度因子体系，覆盖市场主要驱动逻辑：

## ⚙️ 五、数据预处理（关键步骤）

### 1. 去极值（3σ）
限制异常值：

```text
x ∈ [μ - 3σ, μ + 3σ]
```

---

### 2. 标准化（Z-score）

```text
x_std = (x - μ) / σ
```

---

### 3. 行业 + 市值中性化（核心）

对每个交易截面进行回归：

```text
f_i = α + 行业哑变量 + γ·log(mkv_i) + ε_i
```

👉 使用残差 ε 作为新因子值

## 🤖 六、模型构建（XGBoost）

- 模型类型：回归模型
- 预测目标：未来一个月收益率（fwd_return）

### 参数设置

```python
n_estimators = 300
max_depth = 6
learning_rate = 0.05
subsample = 0.8
colsample_bytree = 0.8
```

## 📈 八、策略构建

每月执行以下步骤：

1. 预测所有股票未来收益率
2. 按预测值排序
3. 选取 Top 10 股票
4. 等权配置
5. 持有一个月

👉 属于典型**月度轮动策略**
